Aloia PDF Extractor

User Guide — AI-Powered PDF Data Extraction & Cross-Referencing

What It Does

Aloia PDF Extractor uses AI to read PDF documents and convert their contents into clean, structured data. Upload any PDF, define the fields you need, and receive a downloadable CSV file. Optionally, match records across two datasets using built-in fuzzy matching.

How to Use

Phase 1 — Extract Data
  1. 1

    Define Your Fields

    Specify the data you need extracted. Use the Form Builder to add fields, paste a JSON schema, or describe what you need in plain English.

  2. 2

    Upload PDFs

    Drag and drop one or more PDF files, or click to browse. Multiple documents can be processed at once.

  3. 3

    Review & Download

    View extracted data in a table preview, then download as CSV. Compatible with Excel, Google Sheets, and other data tools.

Phase 2 — Cross-Reference (Optional)
  1. 4

    Upload a Lookup Dataset

    Define a second schema and upload reference PDFs to create a lookup dataset.

  2. 5

    Configure Matching

    Select which columns to match on and which lookup fields to include in the output.

  3. 6

    Download Matched Results

    View matched records and download the combined results as CSV.

Key Details

  • Each row includes a source file reference so you can trace data back to its origin document.
  • Matching is case-insensitive — exact formatting across datasets is not required.
  • All outputs are CSV format, ready for use in any spreadsheet or data application.
